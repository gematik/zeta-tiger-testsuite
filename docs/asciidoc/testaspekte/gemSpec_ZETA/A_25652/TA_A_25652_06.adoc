===== TA_A_25652_06 - Notification Service - Weiterleitung Notification Events an Push Gateways

"Der ZETA Guard Notification Service MUSS Push-Notifications über die von App-Anbietern bereitgestellten Push-Gateways unterstützen, um die Notifications an bestimmte oder alle registrierte Clients eines Anwenders verschicken zu können. Der Notification Service MUSS gemäß [gemF_PushNotification] die Push Konfigurationen von ZETA Clients registrieren und verwalten können und MUSS die vom Resource Server empfangenen **Notification Events** an die dafür **registrierten Push Gateways** der Clients **weiterleiten**. **[<=]**"
