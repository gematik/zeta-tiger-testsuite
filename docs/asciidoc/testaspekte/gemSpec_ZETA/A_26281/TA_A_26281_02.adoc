===== TA_A_26281_02 - PDP - kurzlebige Zertifikate - Signatur - konfigurierbare Gültigkeitsdauer

"Der PDP Authorization Server MUSS für die Signatur von Token Self-signed Zertifikate verwenden.
Die **Gültigkeitsdauer der Signatur-Zertifikate** MUSS in den ZETA Guard-Manifest-Dateien **konfigurierbar** sein. **[<=]"**
