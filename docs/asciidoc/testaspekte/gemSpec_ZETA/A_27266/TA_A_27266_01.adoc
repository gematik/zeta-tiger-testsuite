===== TA_A_27266_01 - PEP HTTP Proxy - Protected Resource Metadata Well-known - Bereitstellung unter spezifischer URI

"Die Komponente **HTTP Proxy MUSS** gemäß [RFC9728] und [opr-well-known.yaml] **ein Well-known JSON Dokument** wie folgt bereitstellen,

+++[...]+++
**GET /.well-known/oauth-protected-resource**
**Host: <FQDN des Resource Servers>**
+++[...]+++
**[<=]**

Anfragen an URI 'https://<Fachdienst>/.well-known/oauth-protected-resource' werden durch Proxy unterstützt.



siehe auch:

<https://datatracker.ietf.org/doc/html/rfc9728/#section-3.1>
