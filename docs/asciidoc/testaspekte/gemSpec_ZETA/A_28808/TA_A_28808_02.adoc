===== TA_A_28808_02 - PDP Authorization Server - Löschfristen - User-Daten Default-Wert

Der Authorization Server (PDP) MUSS konfigurierbare Inaktivitäts-Löschfristen für **User-Daten** durchsetzen und diese Daten nach Ablauf der Fristen löschen. Der **Default-Wert beträgt 1 Jahr**.
