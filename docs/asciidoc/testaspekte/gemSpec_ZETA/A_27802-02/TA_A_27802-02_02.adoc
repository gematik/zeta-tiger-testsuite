===== TA_A_27802-02_02 - ZETA Guard - JWS-Prüfung gemäß RFC7515, Abschnitt 5.2

"Der ZETA Guard MUSS bei der Prüfung von JWT die Prüfschritte gemäß [RFC7519], Abschnitt 7.2, und **[RFC7515], Abschnitt 5.2,** durchführen.

+++[...]+++

**[<=]**"

Hinweise zur konkreten Abdeckung:

* Serialisierung wird korrekt geparst; Protected Header, Payload und Signatur sind base64url-decodierbar.
* Der Protected Header ist gültiges UTF-8-JSON; doppelte Header-Parameter sind unzulässig.
* Erforderliche und durch `crit` angeforderte Header-Parameter müssen verstanden und unterstützt werden.
* Die Signatur wird über den JWS Signing Input geprüft; mindestens eine Signatur muss erfolgreich validieren.
* Das verwendete `alg` muss vorhanden und im Anwendungskontext akzeptabel sein.
