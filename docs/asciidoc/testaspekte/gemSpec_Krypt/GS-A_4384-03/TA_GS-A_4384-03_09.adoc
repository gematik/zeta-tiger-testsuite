===== TA_GS-A_4384-03_09 - TLS-Verbindungen - ZETA Client - Ciphersuiten

Alle Produkttypen, die **Übertragungen mittels TLS** durchführen, MÜSSEN die folgenden Vorgaben erfüllen:

* [...]
* Als **Cipher-Suite** MÜSSEN **TLS_ECDHE_RSA_WITH_AES_128_GCM_SHA256** und **TLS_ECDHE_RSA_WITH_AES_256_GCM_SHA384** unterstützt werden.
* [...]
* Es KÖNNEN weitere Cipher-Suiten aus [TR-02102-2, Abschnitt 3.3.1 Tabelle 1] unterstützt werden.

**[<=]**

Testseitige Konkretisierung für die Featurespezifikation:

* `ecdhe_rsa_aes_128_gcm_sha256` -> `TLS_ECDHE_RSA_WITH_AES_128_GCM_SHA256` -> `(0xC0,0x2F)` -> TLS-Handshake erfolgreich
* `ecdhe_rsa_aes_256_gcm_sha384` -> `TLS_ECDHE_RSA_WITH_AES_256_GCM_SHA384` -> `(0xC0,0x30)` -> TLS-Handshake erfolgreich
