===== TA_A_27016_04 - PoPP-Verifier - technische Rolle oid_popp-token

"Der PoPP-Verifier, der das Signaturzertifikat des PoPP-Token via TI-PKI prüft, MUSS dabei verifizieren, dass das für die Signatur des PoPP-Token verwendete Signaturzertifikat **die technische Rolle oid_popp-token (OID 1.2.276.0.76.4.320) aufweist**."
