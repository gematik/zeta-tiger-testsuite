===== TA_A_27016_05 - PoPP-Verifier - OCSP Status good

"Der PoPP-Verifier, der das Signaturzertifikat des PoPP-Token via TI-PKI prüft, MUSS dabei verifizieren, dass das für die Signatur des PoPP-Token verwendete Signaturzertifikat **per OCSP von der TI-Komponenten-PKI als "good" beauskunftet wird**."
