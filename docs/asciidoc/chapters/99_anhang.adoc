== Übersicht Tiger Proxy Routing

[mermaid,tiger-proxy-routing,svg]
....
include::../diagrams/source/Tiger-routing-diagram.mmd[]
....

Dieses Sequenzdiagramm veranschaulicht das Routing der ZETA Dienste über den Tiger Proxy.
Die Routingkonfiguration sorgt dafür, dass Anfragen zwischen den relevanten Kommunikationsendpunkten im Tiger Proxy protokolliert und der Testsuite über einen separaten Management-Port zugänglich gemacht werden.
Für die Auswertung lädt die Testsuite diese Daten herunter und integriert sie in die Testanalyse.
Zur besseren Lesbarkeit wurde der Tiger Proxy im Diagramm in mehrere Knoten aufgeteilt; im tatsächlichen Deployment existiert jedoch eine zentrale Tiger-Proxy-Instanz, über die der gesamte Verkehr geführt wird.
